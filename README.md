# networking-basic-notes
Catatan dasar networking untuk IT Support &amp; troubleshooting jaringan
## 🌐 1. IP Address

IP Address adalah alamat unik untuk setiap perangkat dalam jaringan.

Contoh cek IP di Windows:
ipconfig

---

## 🌐 2. Ping Test

Digunakan untuk mengecek koneksi ke perangkat lain.

Contoh:
ping 8.8.8.8
ping google.com

---

## 🌐 3. Tracert

Mengetahui jalur koneksi ke tujuan.

tracert google.com

---

## 🌐 4. DNS Check

nslookup google.com

---

## 🌐 5. Port Check

netstat -an
